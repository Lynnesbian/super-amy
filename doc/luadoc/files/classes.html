Tile (name, bg, x, y, hitbox, solid, bouncy, usesStates, states, stateClusters, stateCluster, state)
Parent class for all tiles

Parameters

  • name: The name of the tile
  • bg: Is this a background tile? (boolean)
  • x: X coordinate
  • y: Y coordinate
  • hitbox: The tile's hitbox
  • solid: Whether or not the tile is solid. If it's not, entities will be able to pass through it.
  • bouncy: Whether or not the tile causes entities to bounce off it
  • usesStates: Does this tile have states to animate between?
  • states: A table of states to use
  • stateClusters: Table of stateclusters
  • stateCluster: The current statecluster
  • state: The current state
